## Setup

Clone the Repo

```bash
git clone git@github.com:LangaraVoyagers/project-2-api.git
```

Install the node packages

```bash
npm install
```

Start the server

```bash
npm run dev
```

Open [http://localhost:8000] or the port you configured in the .env
